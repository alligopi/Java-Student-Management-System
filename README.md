# Student Management System (Java)

A simple Java application that collects and displays student details using **Object-Oriented Programming (OOP)** and **Java Collections Framework**.

This program demonstrates:

- Class creation and object instantiation  
- Constructors and methods  
- ArrayList for dynamic data handling  
- Scanner for user input  
- Clean and modular Java code
